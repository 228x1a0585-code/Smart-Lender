# Smart Lender – Loan Approval Prediction System

## Project Overview

The Smart Lender – Loan Approval Prediction System is a machine learning-based web application developed to predict whether a loan application is likely to be approved or rejected based on applicant details. The application automates the preliminary loan eligibility assessment process, helping reduce manual effort and improve decision-making.

---

## Features

* Loan eligibility prediction using Machine Learning.
* User-friendly Flask web application.
* Random Forest model for accurate predictions.
* Simple HTML and CSS interface.
* Real-time prediction results.
* Cloud deployment using Render.

---

---

## Machine Learning Models

The following machine learning algorithms were implemented and evaluated:

* Decision Tree
* Random Forest
* K-Nearest Neighbors (KNN)
* XGBoost

After evaluation, the **Random Forest** model was selected as the final prediction model due to its superior performance.

---


## Application Workflow

1. User enters applicant details.
2. Flask receives the input.
3. Input data is validated.
4. Random Forest model processes the data.
5. Loan approval prediction is generated.
6. Prediction result is displayed to the user.

---

## Deployment

The application has been deployed using the Render cloud platform and is accessible through a web browser.
